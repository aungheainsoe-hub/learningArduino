# 🛠️ Learning Arduino Journey

Welcome to my personal repository dedicated to learning electronics and embedded systems programming using Arduino! This repository serves as a digital logbook for my progress, code sketches, and circuit designs.

Since I am learning hands-on without relying solely on physical hardware initially, most of these projects are designed, wired, and tested using the **Tinkercad Circuits** simulator.

## 🚀 Projects Included

* **01-Blink:** The "Hello World" of hardware. A simple sketch to blink an on-board LED using timing intervals.

## 🧰 Tools & Technologies Used
* **Microcontroller:** Arduino Uno R3 (Simulated)
* **Simulation Platform:** [Autodesk Tinkercad](https://www.tinkercad.com/)
* **Language:** C++ / Arduino Language

## 📂 Repository Structure
Each project is organized into its own folder (or uploaded as standalone files) containing:
* `.ino` file: The actual code script.
* *Future addition:* Wiring diagrams or screenshots of the Tinkercad circuit.
